Extractive RAG Project
Description

This project implements an Extractive Retrieval-Augmented Generation (RAG) system using modern language models and vector databases. It leverages LLaMA 3.1, all-mpnet-base-v2, and ChromaDB to efficiently handle user queries. Features include vector-based query classification, dynamic chunking, and caching for improved performance. The project provides a Streamlit interface for interactive exploration.

Features

Extractive RAG using LLaMA 3.1

Semantic embeddings with all-mpnet-base-v2

Vector storage and retrieval with ChromaDB

Vector-based user query classification

Dynamic chunking of documents for efficient retrieval

Caching of results for faster responses

Interactive Streamlit UI

Technologies Used

Python

LLaMA 3.1 (Language Model)

all-mpnet-base-v2 (Embeddings)

ChromaDB (Vector Database)

Streamlit (Web Interface)
